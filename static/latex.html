When \(a \ne 0\), there are two solutions to \(ax^2 + bx + c = 0\) and they are \[x = {-b \pm \sqrt{b^2-4ac} \over 2a}.\]

When \(a = 0\) and \(b \ne 0\), there is one solution to \(ax^2 + bx + c = 0\) and it is \[x = -c \pm \sqrt{c \over b}.\]

$$a^2+b^2=c^2$$

Note that we cannot simply use \(a^2+b^2=c^2\) to solve for \(a\), \(b\), and \(c\).
